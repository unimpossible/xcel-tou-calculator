# Xcel Time of Use Calculator

Upload your energy bill and calculate Time of Use rate vs flat rate plans.

Streamlit app to parse and show results.

Prefilled with Xcel Energy rates for Colorado.  

## How to run

```bash
pip install -r requirements.txt
streamlit run xcel_tou.py
```

Upload a year's worth of bills and decide which plan is best for you.
